# CryptoPulse

Aplicación móvil e interfaz financiera premium para el monitoreo de criptomonedas en tiempo real (con datos de la API de CoinGecko) y conversor de divisas global.

## Requisitos Previos

Antes de empezar, asegúrate de tener instalado en tu sistema:
- **Node.js** (Versión v24 o superior recomendada)
- **pnpm** (Gestor de paquetes)
- **Android Studio** (Para compilación nativa y Gradle)

##  Instalación y Configuración

1. Clona el repositorio o descarga el proyecto.
2. Instala todas las dependencias necesarias ejecutando en la raíz:
   ```bash
   pnpm install
